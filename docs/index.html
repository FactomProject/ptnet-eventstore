FATIP

This project serves as a reference implementation to demonstrate
a proposed Factom Asset Token standard.

Read a draft of the proposal contained in this repo.

Motivation

Much time and effort as been spent around tokenized value,
this is an attempt to view tokens first and foremost as a meaningful expression of data.

Rather than trying to define a specific token standard, this is an attempt to create an
executible specification that is wholly defined using entries in the Factom blockchain.

The aim of this project is to incubate a standard for publishing information that is:

  • Verifiable while only having access to on-chain data
  • Constructed using well explored math and computer science theory
  • Expressive without being Turing complete
  • Extensible and Updatable using systematic method of expressing behavior
